## Deployment

This repo is connected to **Netlify** for continuous deployment.

Every time you push changes to the `main` branch, the live website updates automatically.

**Live URL:** https://vesastudios.site  
**Netlify URL:** https://whimsical-kashata-cc5a19.netlify.app

---

## How to Update the Website

1. Open `index.html` in a code editor (e.g. VS Code)
2. Make your changes
3. Go to this GitHub repo
4. Click on `index.html` → click the **pencil/edit icon** ✏️
5. Paste your updated code
6. Click **"Commit changes"**
7. Netlify auto-deploys within ~30 seconds ✅
